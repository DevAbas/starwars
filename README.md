# StarWars task assignment
## Running application
The following commands can be executed to run the app locally

### Install packages
Run the following command to install all the dependencies

```bash
npm install 
```
or
```bash
yarn 
```
### Start local server
Running this command will start a local server

```bash
npm start
```
or
```bash
yarn start
```
